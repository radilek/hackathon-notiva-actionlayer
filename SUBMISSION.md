# Last Mile Agent Hackathon submission

## URL

https://github.com/radilek/hackathon-notiva-actionlayer

## Description

PRICE SNIPER turns any product photo into a live US-market buying decision. It identifies the exact variant, researches unfamiliar offers, rejects suspiciously cheap listings using evidence, and recommends the best fair price.

## Explain how you are using ActionLayer

ActionLayer creates and executes a live research-only task for the confirmed product, returning current US listings, prices, seller evidence, returns and marketplace risk signals under a $25 task cap.

## Explain how you are using Novita

Novita Vision identifies the photographed product and variant. A second Novita judgment evaluates only ActionLayer's live evidence, explains risky rejections aloud and selects the best defensible offer.

## Anything else you want to tell us?

Unscriptable input, open judgment, real-world research. A script can sort prices; an agent can decide which price deserves trust.
